js_toolkit_2.0
==============

自用的一个JS工具库，主要用在移动端Html5开发

需要依赖JQuery或者zepto 库

提供的功能有：

1. 地址栏隐藏
2. 触摸事件模块
3. css兼容前缀自动添加
4. 阻止事件触发的方法
5. Cookie操作方法
6. 地址栏参数获取
7. localhost.hash参数获取
8. transform,transition的添加和删除
9. 阻止冒泡方法
10. 旋屏事件方法
11. 弹窗方法
12. 浏览器UA识别
13. Ajax数据获取
14. requestAnimationFrame和cancelAnimationFrame的兼容方法
15. 判断CSS 3D特性的方法
16. APP全屏方法，支持安卓
17. 特性检测，依赖modernizr
